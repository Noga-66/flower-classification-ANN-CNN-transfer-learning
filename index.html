Setup
Convert Keras → TensorFlow.js 
# Install the converter
pip install tensorflowjs

# Convert your trained model
tensorflowjs_converter \
    --input_format=keras \
    saved_models/transfer_learning_flowers.h5 \
    tfjs_model/

# Output → tfjs_model/model.json
#           tfjs_model/group1-shard1of1.bin
💡 Tip: Run this in Google Colab right after training, then download the full tfjs_model/ folder. Upload model.json here — the page will prompt for .bin weights if needed.
